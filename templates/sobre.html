Boas vindas!


Este site foi desenvolvido utilizando Python com o framework Flask e editado no Visual Studio Code, com uma boa dose de café, que é o coração do desenvolvimento!

O objetivo principal do site é servir como um teste das minhas habilidades em programação, permitindo experimentar e aprender com a criação de uma aplicação web funcional.

Além disso, o site também foi criado para agilizar o trabalho de exportar livros do sistema Grupo HNe, facilitando a exportação e organização de dados, tornando o processo mais rápido e eficiente.


Vida longa e próspera.
